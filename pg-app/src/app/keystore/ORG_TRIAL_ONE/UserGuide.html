用戶指南

(English version is at the lower part of this page. 英文版本在本頁下部)

（一）更改電子證書檔案USB內的電子證書密碼

我們建議你把電子證書密碼妥善保管。你也可以使用更改電子證書檔案密碼程式來更改你的電子證書密碼，以便更好地保護你的電子證書檔案USB內的電子證書。

（二）備份電子證書檔案USB內的電子證書

我們建議你把電子證書檔案USB內的電子證書備份至其他儲存媒體。你可參閱香港郵政網頁 https://www.eCert.gov.hk/qstart_c.html.

（三）將電子證書檔案USB內的電子證書匯入至互聯網瀏覽器內

一些網上服務都接受及使用已匯入至互聯網瀏覽器內的電子證書，你可以參閱相關用戶指南內的步驟，將電子證書匯入至不同的互聯網瀏覽器。不過，我們建議你不要把電子證書匯入至公共或他人電腦的瀏覽器內，以防止洩露你的私人密碼匙給他人。

（四）將電子證書檔案USB內的電子證書載入智能身份證內（只適用於電子證書（個人））

如你的智能身份證曾經載入電子證書，你可以參閱相關用戶指南內的步驟，將電子證書檔案USB內的電子證書載入智能身份證內。

（五）使用電子證書於各種網上服務

電子證書可應用於安全電子郵件、網上政府公共服務或網上銀行服務等。你可以瀏覽多種使用電子證書的網上服務，以便於日常網上生活中使用電子證書。

（六）更多關於電子證書及電子證書檔案USB的資訊

你可以瀏覽香港郵政網頁，取得更多關於如何使用電子證書及電子證書檔案USB的資訊。

（五）使用电子证书于各种网上服务

电子证书可应用于安全电子邮件、网上政府公共服务或网上银行服务等。你可以浏览多种使用电子证书的网上服务，以便于日常网上生活中使用电子证书。

（六）更多关于电子证书及电子证书档案USB的资讯

你可以浏览香港邮政网页，取得更多关于如何使用电子证书及电子证书档案USB的资讯。

 

 

User Guide

1. Change PIN for e-Cert on e-Cert File USB

You are advised to keep the PIN of your e-Cert in safe custody. To better protect your e-Cert file stored on e-Cert File USB, you can use the Change Password Program to change your e-Cert PIN.

2. Backup e-Cert on e-Cert File USB

You are advised to make a backup of your e-Cert on e-Cert File USB to other storage medium. You can refer to Hongkong Post web page https://www.eCert.gov.hk/qstart.html.

3. Import e-Cert on e-Cert File USB into Internet Browser

Some online services accept the use of e-Cert loaded into Internet browser. You can follow the steps available in the relevant e-Cert User Guide to import your e-Cert into different Internet browsers. However, you are not advised to import your e-Cert into the browser of public computers or other computers not owned by you to prevent exposure of your private key to others.

4. Load e-Cert from e-Cert File USB to Smart ID Card (Applicable to e-Cert (Personal) only)

If your Smart ID Card has been embedded with an e-Cert before, you can follow the steps available in the relevant e-Cert User Guide to load your e-Cert from e-Cert File USB to Smart ID Card.

5. Use e-Cert in Various Online Services

Hongkong Post e-Cert may be used in services such as secure e-mail, online government services, online banking services, etc. You can explore a range of online services for using e-Cert in your daily online activities.

6. Learn more about e-Cert and e-Cert File USB

You can explore more information on how to use e-Cert and e-Cert File USB from Hongkong Post web site.
